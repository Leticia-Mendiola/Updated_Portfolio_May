Small Business Geocoding

This Web Application was designed as an information hub for downtown small business research. The centerpiece was an interactive map with small business geocoding in which you could click the locations to view information on them, or toggle active & inactive businesses. It also included in-depth reports on historical & current aspects of downtown San Antonio commerce, as well as links for further research.

Day Planner

The purpose of this Web Application was to use as a planner to plan out the user's day. Text could be entered into each time block and then saved using the button at the end of each time block. The web application also utelized Moment.JS to display & track the time, and then each time block would change colors depending on whether that hour was the current hour or had already passed.
